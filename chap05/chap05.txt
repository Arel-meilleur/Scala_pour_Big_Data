    Le chapitre traite des type de données 

   Execice 1
	- Le type de données int stocke des variables entières qui prennent un emplacement mémoire de 4 octets. La valeur du type de données int est comprise entre -2147483648 et 2147483647.
	- Le type de données short stocke la valeur entière des variables qui occupe un emplacement mémoire de 2 octets. La valeur du type de données short est comprise entre -32768 et 32767.
